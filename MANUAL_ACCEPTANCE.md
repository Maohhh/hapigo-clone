# Hapigo Clone Manual Acceptance

## 当前可验收范围

### 1. 产品壳层
- 左侧导航可切换：主页 / 搜索 / 翻译 / 剪贴板 / 设置
- 顶部工具栏存在，整体视觉统一

### 2. 搜索工作台
- 支持搜索应用与文件
- 右侧可查看基础预览信息
  - 类型
  - 大小
  - 修改时间
  - 文本类 snippet
- 底部动作栏支持：
  - 打开
  - 复制内容
  - 访达显示
  - 复制路径

### 3. 命令 / 计算模式
- 在搜索框输入 `=1+2` 这类表达式，可直接得到结果
- 可复制计算结果

### 4. 翻译工作台
- 输入文本翻译
- 截图翻译
- 划词翻译（当前基于剪贴板读取）
- 可复制原文 / 翻译结果

### 5. 剪贴板工作台
- 可查看最近剪贴板列表
- 可查看选中条目的详情
- 可重新复制条目
- 可刷新列表

### 6. 设置页
- 已完成信息架构骨架
- 展示后续配置方向

## 当前仍非最终完善项
- 翻译引擎仍为 mock 结果，不是正式商业翻译服务
- 剪贴板历史当前为基础实现，尚未做到长期持久化数据库
- 更多动作、主题、统计与高级集成仍可继续增强
- Apple Notes / Shortcuts / 1Password 等深度集成尚未正式接入

## 建议验收方式
1. 先整体看 UI 是否达到统一桌面产品感
2. 再逐页测试核心流程
3. 最后列出你希望继续补强的“完整版 2.0”细节
